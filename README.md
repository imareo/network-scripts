### Bash-network-tools

A collection of simple, yet powerful Bash scripts designed for **network administration and automation** on Linux systems. These tools aim to be lightweight, predictable, and highly useful for system administrators and network engineers managing host and router configurations.

---

### Getting Started

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/imareo/bash-network-tools.git
    cd bash-network-tools
    ```

2. **Navigate to Scripts:**
    ```bash
    cd scripts/
    ```

3. **Set Permissions:**
    ```bash
    chmod +x *.sh
    ```
